# FRSH nearby

A Flutter app that connects local food producers with nearby consumers.

Farmers and small food producers list their fresh produce, and consumers
discover and order fresh, locally grown food from producers near them —
shortening the path from farm to table.

## Features

- **For producers** — create a profile, list produce, and manage incoming orders.
- **For consumers** — browse nearby producers, discover fresh local food, and place orders.

## Getting started

```sh
flutter pub get
flutter run
```

Built with [Flutter](https://flutter.dev). Android package / iOS bundle id: `com.frshnearby.app`.
